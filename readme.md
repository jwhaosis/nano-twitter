###To deploy
clone this app from github: https://github.com/jwhaosis/nano-twitter

1. To run this locally, do this in your terminal:
``````bundle install``````
2. To create the schema, run:
`````rake db:migrate`````
3. To seed the data, run:
````rake db:seed````
4. To start the server, run:
````rackup````
